Carbon Emissions Predictor REST Endpoints

This page documents the available endpoints in this Flask application.

Endpoint Method Description
/ GET Serves the documentation page.
/predict POST Predicts carbon emissions using a NuSVR model.

/predict Endpoint

This endpoint accepts a JSON payload containing the following features and predicts the carbon emissions using a pre-trained NuSVR model.

Request body:

  • transport (float): Transportation-related emissions (kg CO2 equivalent).
  • electricity (float): Electricity-related emissions (kg CO2 equivalent).
  • meals (float): Food-related emissions (kg CO2 equivalent).
  • waste (float): Waste-related emissions (kg CO2 equivalent).

Response:

  • result (float): The predicted carbon emissions (metric ton CO2 equivalent).

Error responses:

  • Status code 400 (Bad Request): If any of the required fields are missing in the request body.
  • Status code 403 (Forbidden): If the request is not a POST request or the content type is not JSON.
